# If You Fell Into a Book — Wonderland Quiz

This is a fun, interactive personality quiz inspired by Wonderland.

## How it works

- Users answer 6 questions instinctively.
- Each answer is linked to a playing-card symbol: ♠️ ♣️ ♦️ ♥️
- At the end, the symbol chosen most determines the Reader Identity.
- Result is displayed as a “Reader Identity Card” for users.

## Tech

- HTML, CSS, JavaScript (client-side)
- Hosted on GitHub Pages for free
- Mobile-friendly and interactive
